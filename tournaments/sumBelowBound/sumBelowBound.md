
Given an integer  `bound`, find the maximal integer  `n`  such that  `1 + 2 + ... + n ≤ bound`.

Example

For  `bound = 14`, the output should be  
`sumBelowBound(bound) = 4`.

Input/Output

-   **[execution time limit] 4 seconds (js)**
    
-   **[input] integer bound**
    
    _Guaranteed constraints:_  
    `1 ≤ bound ≤ 5000`.
    
-   **[output] integer**

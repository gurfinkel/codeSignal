
Given an integer  `m`, find the sum of all positive integers  `p`  such that

1.  `2 ≤ p ≤ m`
2.  `p`  and  `m`  are  coprime.

Example

For  `m = 5`, the output should be  
`sumOfCoprimes(m) = 9`.

Explanation:  `9 = 2 + 3 + 4`.

Input/Output

-   **[execution time limit] 4 seconds (js)**
    
-   **[input] integer m**
    
    _Guaranteed constraints:_  
    `5 ≤ m ≤ 120`.
    
-   **[output] integer**

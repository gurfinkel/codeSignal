
Given two integers  `a`  and  `b`, find the remainder of  `a`  when divided by  `b`.

Example

For  `a = 5`  and  `b = 3`, the output should be  
`findTheRemainder(a, b) = 2`.

Input/Output

-   **[execution time limit] 0.5 seconds (cpp)**
    
-   **[input] integer a**
    
    _Guaranteed constraints:_  
    `2 ≤ a ≤ 20`.
    
-   **[input] integer b**
    
    _Guaranteed constraints:_  
    `2 ≤ b ≤ 20`.
    
-   **[output] integer**

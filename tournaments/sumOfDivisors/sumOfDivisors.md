
Find the sum of all divisors of a given integer.

Example

For  `n = 12`, the output should be  
`sumOfDivisors(n) = 28`.

Explanation:  `1 + 2 + 3 + 4 + 6 + 12 = 28`.

Input/Output

-   **[execution time limit] 4 seconds (perl)**
    
-   **[input] integer n**
    
    _Guaranteed constraints:_  
    `1 ≤ n ≤ 15`.
    
-   **[output] integer**

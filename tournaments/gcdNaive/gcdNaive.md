
Calculate the  Greatest Common Divisor(GCD)of two numbers.

Example

For  `a = 4`  and  `b = 6`, the output should be  
`gcdNaive(a, b) = 2`.

Input/Output

-   **[execution time limit] 0.5 seconds (cpp)**
    
-   **[input] integer a**
    
    _Guaranteed constraints:_  
    `1 ≤ a ≤ 500`.
    
-   **[input] integer b**
    
    _Guaranteed constraints:_  
    `1 ≤ b ≤ 500`.
    
-   **[output] integer**
    
    -   _GCD_  of  `a`  and  `b`.

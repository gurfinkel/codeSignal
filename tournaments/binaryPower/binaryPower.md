
Given integers  `n`  and  `k`, find the value of nk  modulo  `107  + 7`.

Example

For  `n = 2`  and  `k = 3`, the output should be  
`binaryPower(n, k) = 8`.

Input/Output

-   **[execution time limit] 4 seconds (py3)**
    
-   **[input] integer n**
    
    A positive integer.
    
    _Guaranteed constraints:_  
    `1 ≤ n ≤ 106`.
    
-   **[input] integer k**
    
    A non-negative integer.
    
    _Guaranteed constraints:_  
    `0 ≤ k ≤ 109`.
    
-   **[output] integer**
    
    -   `nk`  modulo  `107  + 7`

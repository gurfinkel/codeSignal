
Given an integer  `n`, find the value of  `phi(n)`, where  `phi`  is  Euler's totient function.

Example

For  `n = 5`, the output should be  
`eulersTotientFunction(n) = 4`.

Input/Output

-   **[execution time limit] 4 seconds (js)**
    
-   **[input] integer n**
    
    _Guaranteed constraints:_  
    `1 ≤ n ≤ 70`.
    
-   **[output] integer**

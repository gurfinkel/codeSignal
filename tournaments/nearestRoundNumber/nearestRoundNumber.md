
Find the smallest  round number  that is not less than a given  `value`.

Example

For  `value = 122`, the output should be  
`nearestRoundNumber(value) = 130`.

Input/Output

-   **[execution time limit] 4 seconds (js)**
    
-   **[input] integer value**
    
    _Guaranteed constraints:_  
    `10 ≤ value ≤ 130`.
    
-   **[output] integer**

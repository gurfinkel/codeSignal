
Given the array, reverse it.

**Example**

For  `input = [1, 3, 2]`, the output should be  
`myReverse(input) = [2, 3, 1]`.

Input/Output

-   **[execution time limit] 4 seconds (js)**
    
-   **[input] array.integer input**
    
    _Guaranteed constraints:_  
    `1 ≤ input.length ≤ 10`,  
    `1 ≤ input[i] ≤ 100`.
    
-   **[output] array.integer**


Find the largest digit of the given number.

Example

-   For  `n = 111`, the output should be  
    `maxDigit(n) = 1`;
-   For  `n = 132`, the output should be  
    `maxDigit(n) = 3`.

Input/Output

-   **[execution time limit] 4 seconds (js)**
    
-   **[input] integer n**
    
    _Guaranteed constraints:_  
    `0 ≤ n ≤ 2 · 109`.
    
-   **[output] integer**
    
    -   The largest digit of number  `n`.


Find the smallest  prime number  strictly greater than the given  `n`.

Example

For  `n = 7`, the output should be  
`nextPrime(n) = 11`.

Input/Output

-   **[execution time limit] 0.5 seconds (cpp)**
    
-   **[input] integer n**
    
    A positive integer.
    
    _Guaranteed constraints:_  
    `1 ≤ n ≤ 250`.
    
-   **[output] integer**
